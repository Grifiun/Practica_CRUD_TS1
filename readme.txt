Para que el programa funcione es necesario cumplir los siguientes requerimientos:

1. Tener una version de php superior a la 7.0
2. Tener instalado MySQL
3. Tener habilitado el acceso a MySQL desde programas externos (en este caso php).
4. Tener un usuario llamado "phpuser" con password "password".
5. Crear una DATABASE llamada "CRUD_PRUEBA".
6. Crear una tabla llamada Estudiantes con los atributos dpi, nombre y carrera.

Para ejecutar los pasos 5 y 6 puede ejecutar el archivo de "tablas mysql.txt"
